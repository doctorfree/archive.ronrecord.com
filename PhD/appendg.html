Appendix G

The Endo Software Laboratory Diskettes

This appendix describes the two 3.5" diskettes which accompany this document. One diskette is in Unix tar format and the other in MS-DOS format. Each diskette contains the source for the endo program as well as instructions for building and installing it. Recall, endo is an X11 graphical client written in the C programming language. As such, C and X11 development systems are required to build it. Endo was developed using the SCO ODT 3.0 development environment.
